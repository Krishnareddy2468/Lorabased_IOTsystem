# Project Images

This directory contains project documentation images organized by category.

## Folder Structure

| Folder | Contents |
|--------|----------|
| `hardware/` | Photos of ESP32 boards, sensors, LoRa modules, circuit connections, and full system setup |
| `dashboard/` | Screenshots of the web dashboard, charts, login page |
| `architecture/` | System architecture diagrams, data flow diagrams |
| `demo/` | Demo day photos, presentation slides |
| `results/` | ML model accuracy charts, prediction results, sensor data graphs |

## Naming Convention

Use descriptive, lowercase filenames with underscores:
- `esp32_forest_node_setup.jpeg`
- `dashboard_live_data.jpeg`
- `lora_communication_test.jpeg`
